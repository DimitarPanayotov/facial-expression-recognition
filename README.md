# Facial Expression Recognition

Machine learning project for FMI course. Classifies facial expressions into 7 emotions using the FER2013 dataset.

## Team
- Dimitar Panayotov 
- Stefan Tabaakov 

## Setup
```bash
pip install -r requirements.txt
```

## Dataset
Download FER2013 from [Kaggle](https://www.kaggle.com/datasets/msambare/fer2013) and extract into `data/` folder.
